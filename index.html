1. Set Up 🖥

  1. Abre Sublime Text Editor. Se te debe abrir una pantalla negra sin nada escrito.
  2. Crea un folder con el nombre_de_tu_website en tu Escritorio. Arrastra este folder en tu ventana de Sublime Text Editor.
  3. En la navegación izquierda de Sublime Text Editor:
    • Click derecho en el folder nombre_de_tu_website. Selecciona la opción "Nuevo documento"
    • Guárdalo como index.html con Enter o Cmd + S
    • Haz lo mismo y crea otro documento llamado style.css
    • Luego, en nombre_de_tu_website, crea un "Nuevo Folder" || New Folder" llamado imagenes
  4. Finalmente, haz doble click en tu documento index.html en el escritorio para abrirlo con Chrome/Firefox

2. HTML 💀

  1. Vamos a crear la estructura de tu sitio. De esto dependerá como lucirá el resto del diseño de tu página así que tómate tu tiempo en decidir qué va donde, incluso dibujandolo en una página de papel. Si quieres ideas, este es el ejemplo en el que estamos trabajando..
  2. Primero vamos a empezar a programar el HTML en tu index.html. Debería lucir algo así:
  3. Una vez termines la estructura, buscaremos las imágenes que agregaremos al sitio.
    • Para guardar tiempo, puedes solamente hacer click con el botón derecho y guardar estos GIFs en tu folder llamado imagenes para darle movimiento a tu página:
    • Si quieres agregar tus propios GIFs, puedes seguir estos pasos:
      1. Accede a Giphy.
      2. Una vez encuentras el GIF que quieres, haz click derecho en la imágen y guárdala dentro de nombre_de_tu_website/images
      3. Para este momento, te debe salir el GIF detro de tu folder imagenes en Sublime Text Editor. Una vez la encuentras, utiliza su nombre para referirte al GIF en el atributo src de tu <img> tag.
    • Para encontrar íconos puedes acceder aquí: IconStore o The Noun Project.
    • También puedes agregar imágenes tuyas que tengas en tu computadora. Para hacer eso, solo debes asegurarte de tener la imágen que quieres guardada bajo tu folder nombre_de_tu_website/imagenes

3. CSS [fonts & colores] 💅🏼

  1. Conecta el stylesheet style.css en la sección <head> de tu index.html

  2. Luego, copia + pega el código de CSS que encuentras aquí en tu style.css y hazlo tuyo! 🔥

    3. Extra 😏

    • Para buscar tipos de letra: Google Fonts
    • Para buscar inspiración para parejas de fonts: FontPair
    • Para saber que tipos de font usan las páginas que más te gustan: FontFace Ninja
    • Para buscar colores & color palettes: Coolors o Color Hunt
    • Para buscar colores en otros websites, puedes instalar: Colorzilla Chrome plugin

4. Dale nombres + vida a tus elementos gracias a los <div>s 🙆🏻‍♀️

  1. Primero agrupa las distintas secciones con <div>s

  2. Agrégale el atributo id a tu logo, banner o footer- por ej: <div id="banner"> y <div id="footer">

    • Recuerda, id es para elementos que sabes que NO vas a repetir.

    • Si estas pasando problemas con centrar tu texto, este artículo te puede ayudar con eso 😉

  3. Nombra el resto de tus <div>s con classes- por ej: <div class="card">

  4. Una vez tengas terminada la estructura en tu index.html, agregale tu diseño en tu style.css. Aquí te paso par de ideas...

    5. Extra - encuentra una foto cool para agregarle a tu banner

    • Puedes encontrar una gratis de buena calidad aquí: Unsplash o Pexels

5. Fontawesome 💡

  1. Primero, conecta Fontawesome en el <head> 🧠 de tu index.html

  2. Después los enseñas donde quieras 🤩

  3. Y me les das estilo par favar....

6. Bootstrap ⚡️

  1. Conecta Bootstrap en el <head> de tu index.html antes de conectarlo con tu style.css

  2. GAME TIME 😈: juega agregando class="btn btn-info" (a tus botones o links) y class="list-inline" (a tus <ul>) para darle estilo rápidamente a tus componentes. Te recomiendo empezar leyendo un poco la documentación de Bootstrap aquí.

    Para una lista completa de todas las clases que puedes usar, aquí hay un cheetsheet para que veas todas las locuritas más que puedes hacer 🙌🏼

8. Extra 💙

Quieres seguir aprendiendo? ⚡️ Te recomiendo:

  • Cuando estes surfeando el internet de ahora en adelante, haz click derecho para inspeccionar el código detrás del diseño que te gustó. Como cualquier lenguage, entre más lo lees, más fácil será escribirlo después.
  • Intenta agregar más funcionalidades a tu página. Trata de personalizarla lo más que puedas y veras con los pequeños obstáculos con los que te enfrentarás. El mejor consejo que te puedo dar: Google is a developer's best friend 💎. Par de ideas 💡:
    • Crea links para otras páginas dentro de tu mismo folder -> tutorial aquí
    • Juega con diferentes herramientas para hacer layouts:
      • Bootstrap Grid: buena documentación aquí
      • Flexbox: tutorial aquí
      • CSS Grid: tutorial aquí
    • Publica tu sitio web online en GitHub Pages-> sigue mi tutorial aquí o sigue este tutorial de YouTube para deploy por medio del Terminal. El Terminal es esa pantalla que normalmente asimilamos con la pantalla de los hackers.
      • Si tienes una computadora Apple, tienes acceso a ese portal automaticamente, por si quieres geekiar un rato y hacerlo por medio de lo que llamamos el 'Command Line' 😛.
      • Si tienes Windows, lastimosamente se requiere un setup más extenso para hacerlo 😕. Podemos hablar de eso a más profundidad luego, si te interesa..
    • Agregar código de CodePen a tu website y disfruta de la comunidad OpenSource 🤘🏻.
  • Sigue jugando con los componentes de Bootstrap --> forms, navbars, button groups, etc.. Usar sus componentes te ayudará a leer más CSS y te ayudará a avanzar en tu diseño más rápido entre más te vayas conociendo las clases disponibles.
  • Lee sobre diseño web [User Experience Design (UX), UI Design (UI), Components] para saber sobre tendencias web en diseño. Para esto, los newsletters de dev.to y Medium son muy buenos y altamente personalizables.
