1. Set Up 🖥

  1. Abre Sublime Text. Se te debe abrir una pantalla negra sin nada escrito.
  2. Crea un folder perfil en tu Escritorio. Arrastra este folder en tu ventana de Sublime Text.
  3. En la navegación izquierda de Sublime Text:
    • Click derecho en perfil "New file"
    • Guárdalo como index.html con Cmd + S or Ctrl + S
    • Haz lo mismo y crea otro file llamado style.css
    • Luego, en perfil, crea un "New Folder" llamado imagenes
  4. Finalmente, haz doble click en index.html para abrirlo con Chrome/Firefox

2. HTML 💀

  • Para guardar tiempo, puedes solamente hacer click con el botón derecho y guardarlas en tu folder llamado imagenes
  • Then start with this HTML code in index.html

Extra - Encuentra tus propios íconos 😛

  • Los puedes encontrar aquí: IconStore o The Noun Project.

3. CSS [fonts & colors] 💅🏼

Conecta el stylesheet style.css en la sección head de tu HTML perfil.html

Luego, copia + pega el codigo de CSS que encuentras abajo en tu style.css y hazlo bello 🔥 (porque así se ve feo 😕)

  • Para buscar tipos de letra: Google Fonts
  • Para buscar colores & color palettes: Coolors o Color Hunt
  • Para buscar colores en otros websites, puedes instalar: Colorzilla Chrome plugin

4. El mundo de los div 🙆🏻‍♀️

Agrupa las distintas secciones con div

5. Dale nombres + vida a tus tags 🔥

  • Nombra tu banner y footer con id- por ej: <div id="banner"> and <div id="footer">
  • Nombra el resto con class- por ej: <div class="card">
  • Luego agregale este código de CSS a tu style.css

Extra - encuentra una foto cool para agregarle a tu banner

  • Puedes encontrar una gratis aquí: Unsplash o Pexels

7. Bootstrap ⚡️

Conecta Bootstrap en la sección head de tu HTML antes de tu style.css

Juega agregando class="btn btn-primary" (a tus botones) y class="list-inline" (a tus íconos) para darle estilo rápidamente a tus componentes
